# Data Sources

Raw records are excluded from this repository.

Use the public Bureau of Transportation Statistics Airline On-Time Performance records:

https://www.transtats.bts.gov/ONTIME/

Data.gov mirror:

https://catalog.data.gov/dataset/u-s-marketing-air-carriers-on-time-performance

Place downloaded monthly files under `data/raw_open/bts_on_time/` before running the path-construction scripts.
